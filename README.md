# 🎨 UI/UX Design System

This repository contains my **Basic UI/UX Design System** created using **Figma**.
The goal of this project is to build a **consistent and reusable design framework** that can be used for creating modern and user-friendly interfaces.

---

## 📌 Project Overview

A design system helps designers and developers maintain **consistency, scalability, and efficiency** in UI design.
This project demonstrates the core elements required to build a **structured and reusable UI framework**.

---

## ✨ Features

This design system includes:

* 🎨 **Color Palette**
  Defined primary, secondary, and neutral colors for consistent branding.

* 🔤 **Typography Styles**
  Structured typography hierarchy including headings, subheadings, and body text.

* 🔘 **Buttons**
  Reusable button components with different states (primary, secondary, disabled).

* 🧩 **Reusable Components**
  UI elements that can be reused across different screens.

* 📏 **Spacing System**
  Standardized margins and paddings for clean and balanced layouts.

* 🖥 **Layout Guidelines**
  Grid-based layout structure to maintain proper alignment and spacing.
