# 🎨 UI/UX Design System

This repository contains my **Basic UI/UX Design System** created using **Figma**. The goal of this project is to build a **consistent and reusable design framework** that can be used to create modern and user-friendly interfaces.

---

# 📌 Project Overview

A **Design System** helps designers and developers maintain **consistency, scalability, and efficiency** when building user interfaces.

This project demonstrates the **core UI elements and design guidelines** required to create a structured and reusable UI framework.

---

# ✨ Features

This design system includes:

### 🎨 Color Palette

Defined **primary, secondary, and neutral colors** to maintain consistent branding across interfaces.

### 🔤 Typography Styles

Structured **typography hierarchy** including headings, subheadings, and body text for readability and visual hierarchy.

### 🔘 Button Components

Reusable **button components** with multiple states:

* Primary
* Secondary
* Disabled

### 🧩 Reusable Components

UI components designed to be reused across different screens for faster development.

### 📏 Spacing System

Standardized **spacing, margins, and padding** to ensure balanced layouts.

### 🖥 Layout Guidelines

Grid-based layout system to maintain proper **alignment and structure**.
